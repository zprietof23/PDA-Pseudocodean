## Erabiltzaile-Pasahitza

erabiltzailea = ikasle     pasahitza = ikasle123


erabiltzailea = irakasle    pasahitza = irakasle123


erabiltzailea = erabiltzailea      pasahitza = ''    (Ez dauka, utzi zuriz)
